## Installation

Using npm:

```
npm i gatsby-plugin-why-did-you-render
```

Using yarn:

```
yarn add gatsby-plugin-why-did-you-render
```

## How To Use

```
// In your gatsby-config.js
module.exports = {
  plugins: ['gatsby-plugin-why-did-you-render']
};
```
